# Screenshots

Drop real app screenshots here as `1.png`, `2.png`, … `N.png`.

The carousel reads the `SCREENSHOTS` manifest in
[`src/data/site.js`](../../src/data/site.js) — add/remove filenames there to
match what you place in this folder. PNG (portrait phone aspect, ~1080×2400)
looks best inside the phone frame.

Until you add real screenshots, the carousel renders styled placeholder frames.
